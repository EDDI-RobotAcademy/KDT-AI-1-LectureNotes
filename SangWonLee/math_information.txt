1. 벡터의 덧셈, 뺄셈

두 개의 3차원 백터 A = [a1, a2, a3]와 B = [b1, b2, b3]가 있다면, 이들의 합 A + B는 [a1 + b1, a2 + b2, a3 + b3]이 됩니다.
마찬가지로, 이들의 차 A - B는 [a1 - b1, a2 - b2, a3 - b3]이 됩니다.
백터의 덧셈과 뺄셈은 교환법칙과 결합법칙이 성립합니다.
즉, A + B = B + A이고, (A + B) + C = A + (B + C)입니다. 그러나 뺄셈은 교환법칙이 성립하지 않습니다. 즉, A - B ≠ B - A입니다.
백터의 덧셈과 뺄셈을 할 때는 두 백터의 차원이 같아야 합니다. 차원이 다른 백터끼리는 더하거나 빼지 못합니다.

2. 행렬의 덧셈, 뺄셈, 곱셈

A + B = [[a11 + b11, a12 + b12], [a21 + b21, a22 + b22]]
마찬가지로, 이들의 차 A - B는 다음과 같습니다.
A - B = [[a11 - b11, a12 - b12], [a21 - b21, a22 - b22]]
행렬의 덧셈과 뺄셈은 교환법칙과 결합법칙이 성립합니다. 
즉, A + B = B + A이고, (A + B) + C = A + (B + C)입니다.
그러나 뺄셈은 교환법칙이 성립하지 않습니다. 즉, A - B ≠ B - A입니다.
곱 AB를 구하려면 A의 열 수와 B의 행 수가 같아야 합니다. 그리고 결과 행렬 AB의 크기는 A의 행 수 x B의 열 수가 됩니다.
AB ≠ BA입니다. 하지만 결합법칙은 성립합니다. 즉, (AB)C = A(BC)입니다.

3. 벡터, 행렬의 차원

벡터의 성분 개수를 의미합니다. 예를 들어, 3차원 벡터는 3개의 성분을 가지고 있습니다.
행렬의 행과 열의 개수를 의미합니다. 예를 들어, 2x3 행렬은 2개의 행과 3개의 열을 가지고 있습니다.
행렬의 차원은 행렬이 나타내는 선형 변환의 입력 공간과 출력 공간의 차원과 관련이 있습니다.
예를 들어, 2x3 행렬은 3차원 공간에서의 한 점을 2차원 공간에서의 한 점으로 변환하는 선형 변환을 나타낼 수 있습니다.(? 이해안됨)

4. 행렬과 벡터 연산을 위한 최소한의 조건

행렬의 덧셈과 뺄셈: 두 행렬의 크기가 같아야 합니다. 즉, 두 행렬의 행 수와 열 수가 각각 같아야 합니다.
행렬의 곱셈: 첫 번째 행렬의 열 수와 두 번째 행렬의 행 수가 같아야 합니다. 결과 행렬의 크기는 첫 번째 행렬의 행 수 x 두 번째 행렬의 열 수가 됩니다.
벡터와 행렬의 곱셈: 벡터를 왼쪽에서 곱할 경우, 벡터의 길이와 행렬의 행 수가 같아야 합니다. 
	벡터를 오른쪽에서 곱할 경우, 벡터의 길이와 행렬의 열 수가 같아야 합니다. 결과 벡터의 길이는 각각 행렬의 열 수와 행 수가 됩니다.
벡터의 내적: 두 벡터의 길이가 같아야 합니다. 결과는 스칼라 값이 됩니다.
벡터의 외적: 두 벡터 모두 3차원이어야 합니다. 결과는 3차원 벡터가 됩니다.

5. 전치 벡터 및 전치 행렬

전치 행렬은 행렬의 행과 열을 바꾼 행렬입니다. 즉, 전치 행렬의 i번째 행은 원래 행렬의 i번째 열이 됩니다.
벡터는 열 벡터와 행 벡터로 구분됩니다. 
열 벡터는 n x 1 크기의 행렬로, n개의 성분이 세로로 배열된 벡터입니다. 반면, 행 벡터는 1 x n 크기의 행렬로, n개의 성분이 가로로 배열된 벡터입니다.
열 벡터의 전치는 해당 벡터를 행 벡터로 변환합니다. 반대로, 행 벡터의 전치는 해당 벡터를 열 벡터로 변환합니다.

6. 다중 행렬

텐서는 다차원 배열로, 스칼라, 벡터, 행렬을 일반화한 개념입니다. 
스칼라는 0차원 텐서, 벡터는 1차원 텐서, 행렬은 2차원 텐서로 볼 수 있습니다. 이보다 더 높은 차원의 배열을 다차원 텐서라고 합니다.
텐서의 각 차원은 축(axis)이라고 불리며, 텐서의 차원 수는 랭크(rank)라고 합니다. 예를 들어, 3차원 텐서는 3개의 축을 가지며, 랭크가 3입니다.


7. 삼각 함수 + 쌍곡선 함수

sinh(x) = (e^x - e^(-x)) / 2
cosh(x) = (e^x + e^(-x)) / 2
tanh(x) = sinh(x) / cosh(x) = (e^x - e^(-x)) / (e^x + e^(-x))
